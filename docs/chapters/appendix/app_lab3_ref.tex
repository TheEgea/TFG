\section*{Student Entry Point}

Students connect via SSH to pfSense's WAN IP; pfSense DNAT forwards TCP~22 directly to
Server-Web:

\begin{lstlisting}[language=bash]
ssh analyst@<pfSense-WAN-IP>
# Password: An@lyst2024
# Lands on: analyst@helix-web:~$
\end{lstlisting}

No offensive tools are required. Students use only standard Unix utilities:
\texttt{grep}, \texttt{awk}, \texttt{less}, \texttt{cat}, \texttt{find},
\texttt{journalctl}.

\section*{Lab Reset Procedure}

To restore the lab to its initial forensic state after a student session:

\begin{lstlisting}[language=bash]
# Restore EVE-NG snapshot to pre-exercise state
# (from EVE-NG host or web interface)
unl_wrapper -a ...    # see web docs for exact snapshot name
\end{lstlisting}

\noindent Alternatively, revert both Server-Web and Server-DB QCOW2 images from
the pre-staged backup stored at \path{/opt/unetlab/addons/qemu/lab3-backups/}.
