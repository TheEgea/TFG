% ---- Common mistakes --------------------------------------------------------
\section{Common Student Mistakes}

\begin{itemize}[leftmargin=1.5em]
  \item \textbf{Not reading flag.txt fully:} Each flag file contains credentials
        leading to the next hop. Students who stop at the flag string miss the pivot.
  \item \textbf{Wrong password quoting:} The \texttt{!} in \texttt{!Bl4kM3s\$}
        is special in bash. Works fine when typed interactively.
  \item \textbf{Using IP instead of hostname for Server:}
        \texttt{ssh blackmesa@192.168.0.29} works but misses the DNS resolution lesson.
  \item \textbf{Modifying pfSense:} Remind students the scope is enumeration,
        not infrastructure modification.
  \item \textbf{Not pivoting from Server to PC1:} Some students reach pfSense and
        stop. Remind them the pfSense flag file contains further information.
\end{itemize}
